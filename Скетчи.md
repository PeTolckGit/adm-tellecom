## 6. **User Flow простыми блоками**:

```
ЭКРАН ВХОДА
    ↓
ДВУХФАКТОРНАЯ АУТЕНТИФИКАЦИЯ (если включена)
    ↓
ГЛАВНЫЙ ЭКРАН (Dashboard)
    ├──► ПЛАТЕЖИ → Выбор суммы → Способ оплаты → Подтверждение
    ├──► УСЛУГИ → Список тарифов → Подробности → Смена тарифа
    ├──► ПОДДЕРЖКА → Новый тикет → Чат → История тикетов
    ├──► НАСТРОЙКИ → Профиль → Безопасность → Уведомления
    └──► ДОКУМЕНТЫ → Счета → Договоры → Акты
